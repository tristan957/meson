## New Option Overrides

The following options can now be used in `subproject(default_options:)`:

- `b_pch`
- `b_lto`
- `b_lto_threads`
- `b_lto_mode`
- `b_thinlto_cache`
- `b_thinlto_cache_dir`
- `b_sanitize`
- `b_lundef`
- `b_asneeded`
- `b_pgo`
- `b_coverage`
- `b_colorout`
- `b_ndebug`
- `b_staticpic`
- `b_pie`
- `b_bitcode`
- `b_vscrt`
